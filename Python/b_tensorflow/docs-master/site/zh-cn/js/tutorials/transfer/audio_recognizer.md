# 音的移学

在个教程中，将会学到如何建立一个客制化的音分器。将使用Tensorflow.js在器中行。将使用它通出声音来控制器中的滑。

将会使用移学建立一个模型，个模型使用少的数据短音行分。将会使用模型[音命令](https://github.com/tensorflow/tfjs-models/tree/master/speech-commands)。将会在模型的基上一个新的模型，以自己定的声音。

个教程在codelab上已准好了 [从里打codelab！](https://codelabs.developers.google.com/codelabs/tensorflowjs-audio-codelab/index.html)
