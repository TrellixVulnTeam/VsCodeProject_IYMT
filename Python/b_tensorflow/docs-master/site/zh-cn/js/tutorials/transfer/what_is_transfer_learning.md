# 什是移学

的深度学模型通常有数百万的参数（weights），从模型需要大量算源。移学将已在相任中的模型的一部分用到新模型中，从而很大程度地降低大量算源的需要。

例如，本的下一个指南将向展示如何利用一个的已好的模型来建自己的像器，此像器能像中 1000 不同型的物体。使用模型的已有知来的像，会比原始模型所需的数据少得多。

在快速新模型点很有用，尤其是在器和移受限境下。

通常在移学，我不整原始模型的参数，而是除原始模型的最后一，并基于此截断模型的出一个新的（通常相当浅的）模型。就是将在本指南中看到的技。
