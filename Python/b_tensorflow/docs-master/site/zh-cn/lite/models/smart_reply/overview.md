## 怎行

模型会聊天消息生成回建。

内置的模型可以来以下：
<ul>
  <li>行快速：模型内置在中和无需网接。因此，模型推理快速，同平均延只有几毫秒。</li>
  <li>源高效：模型在中占用的内存很小。</li>
  <li>私保：用数据从不。</li>
</ul>
