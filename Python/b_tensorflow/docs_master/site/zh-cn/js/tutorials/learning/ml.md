## 数学概念

机器学是一需要一定数学基的学科，如果只是需要使用机器学模型，那理解模型背后的数学原理是没有必要的。如果整机器学模型或是从建立一个模型，那熟悉基本的数学概念是有助的。
不必先学所有的数学知，而是可以在遇到不熟悉的概念找并学它。
